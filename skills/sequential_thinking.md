# Sequential Thinking Skill — Multi-Step Analytical Reasoning

> **Agent directive**: Use this skill to decompose complex problems before taking actions.
> It allows for revisiting assumptions and branching thought processes.

---

## §1 — The Thinking Process

1. **Step-by-Step**: Each thought follows from the previous one.
2. **Dynamic Adjustment**: Update the total number of steps as the problem unfolds.
3. **Correction & Revision**: If an error is spotted, go back and revise the thinking path.
4. **Branching**: Explore multiple hypotheses in parallel.

---

## §2 — Usage Guidelines

1. **Start Early**: Initiate sequential thinking when a task is first presented if it spans multiple domains.
2. **Review Assumptions**: Explicitly state what you believe is true and verify it.
3. **Conclusion**: Always synthesize the findings into a concrete action plan.

---

## §3 — Common Patterns

- **Debugging**: Trace execution flow and state changes step-by-step.
- **Architectural Design**: Weigh pros and cons of different structural approaches.
- **Compliance Audit**: Systematic verification of code against a set of rules.
